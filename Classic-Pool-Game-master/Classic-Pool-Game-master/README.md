<h3>Keys:</h3>
* 'W' : Increase shot power.<br>
* 'S' : Decrease shot power.<br>
* 'Esc' : Return to main menu.<br>

<h3>Mouse:</h3>
* Left click : shoot.<br>
* Aim by moving the mouse.<br>
